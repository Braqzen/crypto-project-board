# Crypto Project Board

A simple table of projects, nothing more.

Seemingly suspicious projects are not added but otherwise projects are not vetted, DYOR.

## Usage

**1.** Use the search bar to find a project \
**2.** Use the filter dropdown to filter by project category \
**3.** Click on a category chip in the table to filter by category

## Contributing

**1.** Probably won't accept any code \
**2.** Add a project by forking the repo and updating [projects.json](./projects.json) then PR into the repo
